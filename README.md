## Installation

1. Clone the repo

```bash
pip install foobar
```

2. Start redis

```bash
redis-server
```

3. cd API directory

```bash
dotnet watch run
```

4. cd client directory

```bash
ng serve
```

5. Go https://localhost:4200/
